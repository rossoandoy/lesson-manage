# lesson-manage

塾向けのコマ組・出欠・回数報告を管理する Google スプレッドシート + GAS (Google Apps Script) システム。
スプレッドシートのカスタムメニューとサイドバーから、ブース表の作成・講師/生徒の配置・出欠記録・回数報告の生成までを一元管理できます。

---

## シート一覧

| シート名 | 役割 | 作成方法 |
|---------|------|---------|
| **ブース表** | コマ組グリッド（日付 × ブース × 時限） | サイドバーの年度グリッド生成 or メニュー |
| **印刷シート** | 授業データベース（1行 = 1コマ） | ブース表から自動同期 |
| **回数報告** | 生徒別・月別の出欠集計レポート | メニューから生成 |
| **回数報告_〇〇** | 全生徒一括レポート時の個別シート | 一括生成時に自動作成 |
| **データ出力** | フィルタ付きデータエクスポート | メニューから生成 |
| **tran** | SF 連携キャッシュ（請求データ） | メニュー「tran シートを初期化」or SF 同期 |
| **master_students** | 生徒マスタ（ID / 氏名 / 学年） | メニュー「生徒マスタを初期化」 |
| **master_staffs** | 講師マスタ（ID / 氏名） | メニュー「講師マスタを初期化」 |
| **master_subjects** | 教科マスタ（教科名一覧） | メニュー「教科マスタを初期化」 |
| **Template_Cover** | 表紙（教室名・教室長名・バージョン・更新日） | 管理メニュー「Admin シートを初期化」 |
| **Admin_Version** | バージョン管理（リリース履歴） | 管理メニュー「Admin シートを初期化」 |
| **Admin_Classrooms** | 教室一覧・プロビジョニング管理 | 管理メニュー「Admin シートを初期化」 |

## ブース表の見方

ブース表はグリッド形式で授業コマを管理します。

```
          │  1限目(5列)  │  2限目(5列)  │  3限目 ...
          │ブース 講師 生徒 学年 教科│ブース 講師 生徒 学年 教科│ ...
──────────┼──────────────┼──────────────┼─────────
4/1 (火)  │ 1    山田  鈴木  中2  数学│ 1    佐藤  田中  高1  英語│
          │            高橋  中3  国語│           加藤  中1  理科│
          │ 2    ...              │ 2    ...              │
          │                       │                       │
          │ 3    ...              │ 3    ...              │
          │ ...                   │ ...                   │
──────────┼──────────────┼──────────────┼─────────
4/2 (水)  │ ...                   │ ...                   │
```

- **縦方向**: 日付ごとにブース数 × 2行（1:2 授業対応で生徒2名分）
- **横方向**: 時限ブロック（各5列: ブース番号 / 講師 / 生徒 / 学年 / 教科）
- ブース番号・講師名は 2行結合（1:2 でも同一ブース・講師）

## メニュー操作ガイド

スプレッドシートを開くと「**コマ組メニュー**」が表示されます。

| メニュー項目 | 説明 |
|-------------|------|
| コマ組サイドバーを開く | 設定・コマ組操作用のサイドバーを表示 |
| ブース表を初期化（表示期間を設定） | 開始日〜終了日を入力してグリッドを生成 |
| 回数報告を生成 | 印刷シートの出欠データから生徒別月別レポートを作成 |
| 全生徒レポート一括生成 | 全生徒分の回数報告シートを一括作成 |
| 前年度累計を設定 | 前年度末の出欠累計を回数報告に反映 |
| 教科マスタを初期化 | 教科一覧シートをデフォルト値で作成 |
| 講師マスタを初期化 | 講師マスタシートのヘッダーを作成 |
| 生徒マスタを初期化 | 生徒マスタシートのヘッダーを作成 |
| tran シートを初期化 | SF 連携用キャッシュシートを作成 |
| 印刷シートヘッダーを初期化 | 印刷シートの列ヘッダーを設定 |
| 印刷シートを同期 | ブース表の全データから印刷シートを再生成 |
| データ出力（フィルタ付き） | 生徒名・講師名・期間でフィルタしてデータ出力シートに書き出し |
| ブース表を印刷用に表示 | 指定期間のブース表を印刷向けに整形表示 |

### 管理メニュー（親シート用）

Admin シートが存在するスプレッドシートでは「**管理メニュー**」が追加表示されます。

| メニュー項目 | 説明 |
|-------------|------|
| 教室一覧を同期 (SF→シート) | Salesforce から教室データを取得し Admin_Classrooms に反映 |
| 新規教室のシートを生成 | テンプレートをコピーして教室用 SS を作成（Phase 5 で実装） |
| 全教室テンプレート更新 | 全教室の SS をテンプレートで更新（Phase 5 で実装） |
| SF URL書き戻し | Admin_Classrooms の SS URL を Salesforce に書き戻し |
| tran シート同期 (SF→シート) | 指定年月の請求データを SF から取得し tran シートに書き込み |
| SF 接続設定 | Salesforce の接続情報（Instance URL / Client ID / Secret）を設定 |
| SF 接続テスト | Salesforce への接続を確認 |
| バージョン情報を表示 | 現在のテンプレートバージョンを表示 |

Admin シートが存在しない場合は「**Admin シートを初期化（親シート化）**」メニューが表示され、実行すると Admin_Classrooms / Admin_Version / Template_Cover の 3 シートが作成されます。

## サイドバーの使い方

「コマ組サイドバーを開く」で表示されるサイドバーには 2 つのタブがあります。

### 設定タブ

| 項目 | 内容 |
|------|------|
| 教室名 | 教室の表示名を設定 |
| ブース数 | 1日あたりのブース数（デフォルト 5） |
| 時限数 | 1日あたりの時限数（デフォルト 10） |
| 時間帯 | 各時限の開始〜終了時刻 |
| 年度グリッド生成 | 年度（例: 2025）を指定して4月〜3月のグリッドを一括生成 |
| 表示期間 | ブース表に表示する日付範囲を絞り込み |
| 表示時限 | 表示する時限範囲を絞り込み |

### コマ組タブ

| 機能 | 説明 |
|------|------|
| コマ確認 | ブース表でセルを選択すると、そのコマの講師・生徒・時限を表示 |
| 講師配置 | 講師を選択して繰り返し（毎週等）配置 |
| 生徒配置 | 生徒・学年・教科を選択してコマに配置 |
| 出欠記録 | 出席 / 欠席 / 振替を記録（振替先日付も指定可） |
| 検索 | 生徒名でブース表を検索しハイライト表示 |
| 一括削除 | 日付指定 / 生徒指定 / 講師指定で複数コマを一括削除 |

## 基本的な使い方

1. **マスタを初期化する**
   - メニューから「教科マスタを初期化」「講師マスタを初期化」「生徒マスタを初期化」を実行
   - 講師マスタ・生徒マスタにデータを入力する

2. **教室設定をする**
   - サイドバーの設定タブで教室名・ブース数・時限数・時間帯を設定

3. **年度グリッドを生成する**
   - サイドバーの設定タブで年度を入力し「年度グリッド生成」を実行

4. **講師を配置する**
   - ブース表で配置先のセルを選択
   - サイドバーのコマ組タブで講師を選択し配置（繰り返し設定も可）

5. **生徒を配置する**
   - 講師が配置済みのコマに対して生徒・教科を配置

6. **出欠を記録する**
   - サイドバーのコマ組タブで出席 / 欠席 / 振替を記録

7. **回数報告を生成する**
   - メニューから「回数報告を生成」で生徒別月別の集計レポートを作成

## 印刷シートの列構成

印刷シートは授業データベースとして機能します（ヘッダー: 8行目、データ: 9行目〜）。

| 列 | 項目 | 説明 |
|----|------|------|
| A | 日付 | 授業日 |
| B | 曜日 | 自動設定 |
| C | 時限 | 時限番号 |
| D | ブース | ブース番号 |
| E | 講師 | 講師名 |
| F | 生徒 | 生徒名 |
| G | 学年 | 生徒の学年 |
| H | 教科 | 教科名 |
| I | 定員 | 1:1 または 1:2 |
| J | 出欠 | 出席 / 欠席 / 振替 |
| K | 振替元日付 | 振替で休んだ元の授業日 |
| L | 振替先日付 | 振替先の授業日 |

## Salesforce 連携

管理メニューから Salesforce と連携できます。初回は接続設定が必要です。

### 接続設定手順

1. **管理メニュー → SF 接続設定** を実行
2. Instance URL を入力（例: `https://your-domain.my.salesforce.com`）
3. Connected App の Client ID を入力
4. Client Secret を入力
5. **管理メニュー → SF 接続テスト** で接続を確認

### 連携機能

| 機能 | 説明 |
|------|------|
| 教室一覧を同期 | SF の Account (Location) から教室データを取得し Admin_Classrooms にアップサート |
| SF URL書き戻し | Admin_Classrooms の SS URL を SF の Account.Spreadsheet_URL__c に書き戻し |
| tran シート同期 | 指定年月の MANAERP__Invoice__c データを tran シートに一括書き込み |

**前提条件**:
- SF 側に Connected App が設定済み（Client Credentials フロー）
- `Spreadsheet_URL__c` は SF Account に作成が必要（URL 書き戻し時）
