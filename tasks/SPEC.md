## Phase 0

### P0.1 initStudentMaster()

**要件源**: 既存 initStaffMaster() との対称性
**変更ファイル**: `01_Main.gs`
**関数シグネチャ**:
```javascript
function initStudentMaster() → void
```
**動作**:
1. `master_students` シートを getOrCreateSheet で取得/作成
2. ヘッダー行（行1）に `ID | 氏名 | 学年` を設定（太字）
3. toast で完了通知
4. onOpen メニューに追加

**テスト**: GAS エディタから手動実行 → シート作成・ヘッダー確認
